# Scripts

## update-passwords.js

Actualiza las contraseñas de todos los participantes en Firestore.

**Patrón:** `nombre + número` (ej: `laura100`, `lucia101`, etc.)

**Ejecutar:**
```bash
node scripts/update-passwords.js
```

**Contraseñas que asigna:**
- Laura → `laura100`
- Lucía → `lucia101`
- Olivia → `olivia102`
- Eva → `eva103`
- Pablo → `pablo104`
- Lucas → `lucas105`
- Darío → `dario106`
- Elena → `elena107`
- Javi → `javi108`
- Nic → `nic109`
- Jose M. → `josem.110`
- Charo → `charo111`
- Abuelo → `abuelo112`

**Importante:**
1. Ejecutar desde la carpeta raíz del proyecto
2. Necesita conexión a Firestore
3. Actualiza todos los participantes en una sola ejecución
